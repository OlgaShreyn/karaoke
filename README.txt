Задача: караоке
Запускаемый файл - kar.py
Программа имеет графический интерфейс.
Текст песни выводится на фоне, который по умолчанию менятся случайно каждые 10 секунд
Программу имеет несколько видов запуска
1. Без параметров, тогда для воспроизведения будут предложены файлы из текущей директории
2. После ключевого слова --path требуется передать директорию, в которой находятся файлы
3. После ключевого слова --file требуется передать полный путь до файла, который будет воспроизводиться
4. с помощью флага -с можно сделать фон неизменным
5. После ключевого слова --path_back требуется передать директорию, в которой находятся файлы, подходящие для заднего фона (png).
Или сам файл. По умолчанию файлы для заднего фона расположены в папке backgrounds
С помощью кнопок play и stop воспроизводится/останавливается музыка из выбранного файла кнопки previous/next song служат для переключения песен
из списка в правой части окна. Кнопка начать заново повторяет воспроизведение, текущей песни
